# Airplane Seating (C++)
### Description

A menu-driven C++ console program that assigns seats on an airplane with 13 rows and 6 seats per row. 
Seating is divided into ticket classes:

First Class: Rows 1–2

Business Class: Rows 3–7

Economy Class: Rows 8–13

The program validates seat selection and prevents double booking.

### How to Run (from Visual Studio)
```cl /EHsc Source.cpp```

```./Source.cpp```
